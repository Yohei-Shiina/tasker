# アプリ概要
アプリ名：tasker

Ruby on Rails 5 速習実践ガイドを参考に作成したToDoアプリです。<br>
デプロイはしていません。<br>

<img width="1440" alt="スクリーンショット 2019-06-26 0 44 50" src="https://user-images.githubusercontent.com/35527421/60112931-ae3dd380-97ab-11e9-83e1-757d70de6099.png">



# アプリ機能一覧
- ユーザー作成機能
- タスクの閲覧、作成、編集、削除、検索
- 削除時は非同期通信

# その他
- RSpecテスト実行
- タスク作成時のメール送信機能
